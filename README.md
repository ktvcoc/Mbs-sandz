# Marina Bay Sandz - Giao diện Giải trí Cao cấp

Dự án giao diện web hiện đại cho hệ thống giải trí Marina Bay Sandz, được tối ưu hóa cho trải nghiệm người dùng trên cả Desktop và Mobile.

## 🚀 Tính năng nổi bật
- **Giao diện hiện đại**: Sử dụng tông màu sang trọng, bố cục rõ ràng.
- **Tối ưu Mobile (Responsive)**: Hiển thị hoàn hảo trên mọi kích thước màn hình.
- **Hiệu ứng mượt mà**: Các tương tác người dùng được chăm chút bằng CSS Animation.
- **Cấu trúc nhẹ**: Tải trang nhanh, mã nguồn sạch, dễ dàng chỉnh sửa.

## 📂 Cấu trúc dự án
- `index.html`: Cấu trúc nội dung chính của trang web.
- `style.css`: Quản lý giao diện, màu sắc và hiệu ứng.
- `script.js`: Xử lý các tương tác logic phía người dùng.
- `manifest.json`: Hỗ trợ tính năng Web App (PWA).

## 🛠 Hướng dẫn phát triển
1. Clone repository về máy.
2. Mở file `index.html` trực tiếp trên trình duyệt hoặc sử dụng Live Server.
3. Chỉnh sửa CSS trong file `style.css` để thay đổi màu sắc chủ đạo.
